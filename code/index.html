The Language of Geometry

Warning

This is still unfinished. Many things that you may expect to work are not. In a random order:
  • There are no mathematical operation yet (i.e. 3*4 = ERROR, not 12)
  • Every float needs a dot: 4 and 4. are not equal, nor interchangeable
  • This is still generally rough : the screen is limited, and so are many things.
Generally speaking, if you strictly respect the syntax, you should have no surprise (or you are welcome to report them!)

Syntax

Numberint ::= | 0, 1, 2, ...
Numberfloat ::= | Numberint.Numberint
| Numberint.
Number ::= | Numberint | Numberfloat | Undefined
Program ::= | Program ; Program
| Draw(Numberfloat,Numberfloat)
| Set(Numberfloat,Numberfloat)
| Save(string)
| Load(string)
| Turn(Numberfloat)
| DiscreteRepeat(Numberint) { Program }
| Integrate(Numberfloat) { Program }
| {}
